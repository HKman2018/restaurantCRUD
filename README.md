# 簡介

       網站主要介紹餐廳的資料和食評 

# 安裝方法

    
      . git clone https://github.com/HKman2018/restaurant_list.git
      
      . cd ./restaurant_list
      
      . npm install 
      
      . npm install exress 
      
      . npm install nodemon
      
## 使用方法
   使用者可以增加喜愛的餐廳,可以為餐廳進行更新資料,可以搜尋餐廳
